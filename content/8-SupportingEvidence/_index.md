## Notes for Reviewers

The final AWS workshop is intentionally smaller than the graduation project and the CLEF work. The supporting artifacts explain the learning path and technical background, while the workshop itself focuses on one final path: local Chainlit UI, Lambda Function URL, Lambda container query embedding, Qdrant Cloud retrieval, Gemini answer generation, and AWS observability.

The private graduation project repository is not shared directly. Instead, this page provides a redacted thesis reference and a repository screenshot as supporting evidence.
