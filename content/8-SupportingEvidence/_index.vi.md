## Ghi chú cho reviewer

Workshop AWS cuối cùng được giữ nhỏ hơn so với đồ án tốt nghiệp và công việc CLEF. Các tài liệu hỗ trợ giải thích learning path và nền tảng kỹ thuật, còn workshop tập trung vào một final path: Chainlit UI local, Lambda Function URL, Lambda container query embedding, Qdrant Cloud retrieval, Gemini answer generation, và AWS observability.

Repository đồ án tốt nghiệp vẫn không được chia sẻ trực tiếp. Thay vào đó, trang này cung cấp báo cáo đã redact và screenshot repository như supporting evidence.
